# NovaTech Landing Page 

A clean, modern and fully responsive landing page for a fictional tech startup — built with HTML, CSS and vanilla JavaScript.

## 🔗 Live Demo
https://arinola001.github.io/NovaTech/

## ✨ Features

- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations
- Animated stat counters
- Interactive hamburger menu for mobile
- Clean light & modern design
- Hover effects on feature cards

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript

## 📁 Project Structure

```
NovaTech/
│
├── index.html
└── README.md
```

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Arinola001/novatech-landing-page.git
```

2. Open `index.html` in your browser — no dependencies needed!!
